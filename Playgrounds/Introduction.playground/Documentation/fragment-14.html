Channels and Observables

A Channel is bi-directional access to some underlying state. It is always backed by a reference type, either a class in Objective-C or a reference wrapper around a Swift value type. A Channel is a specialization of a Observable, which provides uni-directional flow of events. Events are not limited to state changes.
